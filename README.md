## Descripción del proyecto

Sistema de inteligencia operativa para automatizar el control presupuestal del área de Facilidades y Flota de una tabacalera multinacional, eliminando el proceso manual mensual de descarga, depuración y revisión de gastos desde el sistema ERP corporativo.
